## 📝 概要

LINEのメッセージを直接Obsidianのノートとして保存できるプラグインです。スマートフォンからの素早いメモ取りとObsidianでの整理を組み合わせることで、効率的なナレッジ管理を実現します。

### 🆕 新アーキテクチャの採用

**重要**: このプラグインは、よりシンプルで安全なアーキテクチャに移行しました。

- 外部サーバー（Firebase）への依存を廃止
- すべてのデータをローカルで直接処理
- プライバシーとセキュリティの強化
- メンテナンスの簡素化

## 🌟 特徴

- 📱 LINEからの直接メモ送信
- 🔒 エンドツーエンドの安全な通信
- 💨 高速なメッセージ処理
- 📂 自動的なMarkdownファイル作成
- ⚡ プラグイン内蔵のローカルサーバー
- 🔐 プライバシー重視の設計

## 🛠️ 動作の仕組み

1. LINEでメッセージを送信
2. Cloudflare Workersがメッセージを受信
3. ローカルのObsidianプラグインが処理
4. Markdownファイルとして保存

## 📥 インストール

1. Obsidianの設定を開く
2. サードパーティプラグイン > コミュニティプラグイン > 閲覧
3. "LINE to Obsidian"を検索
4. インストールをクリック

## ⚙️ セットアップ

1. プラグインを有効化
2. LINEで公式アカウントを友だち追加
3. プラグイン設定画面でポート番号と保存フォルダを設定
4. LINEで認証トークンを確認

## 📝 使い方

1. Obsidianでプラグインを有効化
2. LINEで通常のメッセージを送信
3. 自動的にObsidianのVaultに保存
4. 設定した保存フォルダでメモを確認

## 🔒 プライバシーとセキュリティ

- すべてのデータはローカルで処理
- 中間サーバーでのデータ永続化なし
- 暗号化された通信のみを使用
- ローカルサーバーは127.0.0.1のみアクセス可能

## ⚠️ 注意事項

- プラグイン有効時はローカルサーバーが起動します
- ポート51828（デフォルト）が使用可能である必要があります
- インターネット接続が必要です

## 🐛 トラブルシューティング

### ローカルサーバーが起動しない
- ポートが他のアプリケーションで使用されていないか確認
- プラグインの再起動を試す
- 別のポート番号を設定

### メッセージが届かない
- インターネット接続を確認
- プラグインが有効化されているか確認
- 認証トークンが正しいか確認
